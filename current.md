# 当前关：更新 Todo

预计用时：15–20 分钟。今天只做这一关。

第五关已经通关。第六关把已经学过的路径 ID 和请求 JSON 组合起来：用 `PUT /todos/{id}` 替换一个已有 Todo。

## 通关条件

- 运行 `go run .`
- 用 `PUT /todos/1` 更新标题和完成状态，看到 `200 OK`
- 再用 `GET /todos/1` 确认内存中的 Todo 已经改变
- 亲自修改一次标题，并让空标题返回 `400 Bad Request`
- 能说出：路径 ID、请求 JSON 和 `todos[index]` 如何共同完成更新

## 开始

打开 [第六关课程](lessons/0006-update-todo.html)。

想先看整体方向，可以打开 [Go 后端闯关地图](roadmap.html)。

如果今天只有 2 分钟：只发送一次课程中的 `PUT /todos/1`，再用 `GET` 确认变化，也算保温成功；下一次仍从本关继续。
