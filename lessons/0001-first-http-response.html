让 Go 服务第一次回应你

你最终会让请求经过 Go 和 MySQL 再返回 JSON。这一关只建立第一段反馈回路：代码 → 启动 → 请求 → 响应。

本关胜利画面

浏览器里出现 Hello, Go，然后你亲手把它改成另一句话。

1. 启动

在这个学习目录打开终端，运行：

go run .

看到“服务已启动”后不要关闭这个终端。它现在正在等待请求。

2. 发一个请求

用浏览器打开 http://localhost:8080/hello。如果看到 Hello, Go，第一半已经完成。

3. 只读懂这一小段

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, Go")
}

helloHandler 是处理请求的函数。现在不用背参数；只要知道 fmt.Fprint 把文字写进响应。

4. 亲自改一处

  1. 打开 main.go。
  2. 把 "Hello, Go" 改成一句你自己的话。
  3. 回到终端按 Control + C 停止服务，再运行一次 go run .。
  4. 刷新浏览器，确认新文字出现。

5. 一题验收

访问 /hello 后，哪一个函数负责写回文字？

向老师通关

回复你改成的文字，并用一句话说说 helloHandler 做了什么。老师确认后再解锁下一关。

卡住时直接问老师，把终端中的完整报错贴出来即可。速查：第一次 HTTP 回路。主要资料：Go 官方 net/http 文档。
