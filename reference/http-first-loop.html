Go 新手速查

第一次 HTTP 回路

浏览器发请求
→ handler 处理
→ 写回响应

目前只需要认出三处

http.HandleFunc("GET /hello", helloHandler) // 把路径交给函数
func helloHandler(...) { ... }              // 处理请求
http.ListenAndServe(":8080", nil)           // 在 8080 端口等待

两个命令

go run .
curl http://localhost:8080/hello

先不要背参数类型。你只需能沿着路径找到处理函数，再找到它写回的文字。
